<p>My group got two laser sensors working at once. To do so, we disable the lasers we don't care about by: setting the SHDN pin to LOW. (Note: When setting SHDN pin to HIGH/enabling the laser, make sure to call begin() again.)</p>
<p>1. We got this library to work with one sensor: <a href="https://github.com/adafruit/Adafruit_VL6180X">https://github.com/adafruit/Adafruit_VL6180X</a> (There's a direct download on the adafruit website: <a href="https://learn.adafruit.com/adafruit-vl6180x-time-of-flight-micro-lidar-distance-sensor-breakout/wiring-and-test">https://learn.adafruit.com/adafruit-vl6180x-time-of-flight-micro-lidar-distance-sensor-breakout/wiring-and-test</a>)</p>

<p>// inspiration from site: http://arduino.stackexchange.com/questions/25050/multiple-vl6180x-distance-sensors-interactive-problem<br /><br />/* Multiple laser procedure:<br />&nbsp;*&nbsp; 1. Wire SHDN (SHutDowN) pins.<br />&nbsp;*&nbsp; 2. Disable ALL lasers. (write SHDN pin &gt; LOW)<br />&nbsp;*&nbsp; 3. Enable 1 laser. (write SHDN pin &gt; HIGH &amp; call laser.begin())<br />&nbsp;*&nbsp; 4. Read value.<br />&nbsp;*/<br />/* Note: <br />&nbsp;*&nbsp; For Arduino Uno, SDA = A5 &amp; SCL = A4<br />&nbsp;*&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; for other boards, see: https://www.arduino.cc/en/Reference/Wire<br />&nbsp;*/<br /><br />#include "Adafruit_VL6180X.h"<br /><br />Adafruit_VL6180X lasers;<br />&nbsp; byte laser1SHDNPin = 7; // SHutDowN pin<br />&nbsp; byte laser2SHDNPin = 6;<br />//&nbsp; byte laser3SHDNPin = 3;<br /><br />void setup() {<br />&nbsp; pinMode(laser1SHDNPin, OUTPUT);<br />&nbsp; pinMode(laser2SHDNPin, OUTPUT);<br />//&nbsp; pinMode(laser3SHDNPin, OUTPUT);<br />&nbsp; digitalWrite(laser1SHDNPin, LOW); // Disable laser1<br />&nbsp; digitalWrite(laser2SHDNPin, LOW); // Disable laser2<br />//&nbsp; digitalWrite(laser3SHDNPin, HIGH); // Disable laser<br /><br />&nbsp; Serial.begin(9600);<br />}<br /><br />void loop() {<br />&nbsp; // Enable laser 1<br />&nbsp; digitalWrite(laser2SHDNPin, LOW);&nbsp; // Disable laser2 before...<br />&nbsp; digitalWrite(laser1SHDNPin, HIGH); // enabling laser1<br />&nbsp; // Print laser 1 value<br />&nbsp; Serial.print("laser1: ");<br />&nbsp; lasers.begin();<br />&nbsp; Serial.print(lasers.readRange());<br /><br />&nbsp; // Enable laser 2<br />&nbsp; digitalWrite(laser1SHDNPin, LOW);&nbsp; // Disable laser1 before...<br />&nbsp; digitalWrite(laser2SHDNPin, HIGH); // enabling laser2<br />&nbsp; // Print laser 2 value<br />&nbsp; Serial.print("\t\tlaser2: ");<br />&nbsp; lasers.begin();<br />&nbsp; Serial.println(lasers.readRange());<br />&nbsp; delay(500);<br />}</p>
